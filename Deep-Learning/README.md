# Deep-Learning

Welcome to the **Deep-Learning** directory! This folder is dedicated to my hands-on practice, experiments, and learning journey with machine learning and deep learning concepts.

## 📂 Contents

* **`DL practice.ipynb`**: A Jupyter Notebook containing general practice exercises, covering various deep learning implementations and code experiments.
* **`TensorCoreConcepts.ipynb`**: A notebook specifically focused on the fundamental building blocks of TensorFlow, including tensor operations, shapes, and manipulations.
* **`tensorflow`**: A plain text log/note detailing my progress. (e.g., *Day 12: First steps into TensorFlow and building the foundation for advanced studies!*)

## 🛠️ Prerequisites

To run the notebooks in this directory, you need Python installed along with Jupyter and TensorFlow.

You can typically install the required libraries like this:
```bash
pip install notebook tensorflow pandas numpy matplotlib
```

## 🚀 How to Run

1. Open your terminal and navigate to this directory.
2. Start the Jupyter Notebook server:
   ```bash
   jupyter notebook
   ```
3. A web browser window will open. Click on either `DL practice.ipynb` or `TensorCoreConcepts.ipynb` to view the code, read the explanations, and execute the cells step by step.
